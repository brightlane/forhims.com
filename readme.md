## How it works

`build.py` runs daily via GitHub Actions and generates all HTML pages, `sitemap.xml`, `robots.txt`, and `llms.txt` from scratch, then commits and pushes to main. GitHub Pages deploys automatically on every push.

## Files

| `.github/workflows/hims-engine.yml` | Daily build trigger (3 AM UTC) |
